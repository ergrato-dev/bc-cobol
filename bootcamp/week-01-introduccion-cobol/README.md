# Semana 01: Introducción a COBOL

## 🎯 Objetivos

Al finalizar esta semana, el estudiante será capaz de:

- Comprender la historia y relevancia actual de COBOL en la industria
- Instalar y configurar GnuCOBOL 3.2+ en su sistema Linux
- Identificar las 4 divisiones de un programa COBOL y su propósito
- Escribir, compilar y ejecutar su primer programa "Hola Mundo" en COBOL
- Usar DISPLAY y ACCEPT para entrada/salida básica
- Comprender el formato de columnas (fixed vs free format)

## 📚 Contenido

### 1-teoria/

- **historia-cobol.md**: Origen en 1959, evolución, estándares (68, 74, 85, 2002, 2014, 2023)
- **por-que-cobol-hoy.md**: Banca, seguros, gobierno, mainframes, ~220B líneas activas
- **instalacion-gnucobol.md**: Fedora, Ubuntu, WSL, verificación
- **estructura-programa.md**: Las 4 divisiones explicadas (IDENTIFICATION, ENVIRONMENT, DATA, PROCEDURE)
- **formato-columnas.md**: Área A/B, fixed vs free format, comentarios

### 2-practicas/

- **hola-mundo.cbl**: Primer programa COBOL con DISPLAY
- **entrada-salida.cbl**: Uso de ACCEPT y DISPLAY
- **formato-libre.cbl**: Comparación fixed vs free format

### 3-proyecto/

- **starter/presentacion.cbl**: Programa que solicita nombre, edad, ciudad y genera presentación formateada

- Proyecto: 3h
