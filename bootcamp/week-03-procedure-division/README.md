# Semana 03: PROCEDURE DIVISION

## 🎯 Objetivos

Al finalizar esta semana, el estudiante será capaz de:

- Usar los verbos fundamentales MOVE, COMPUTE, DISPLAY y ACCEPT
- Implementar control de flujo con IF/ELSE y EVALUATE
- Escribir condiciones compuestas con AND/OR/NOT
- Crear párrafos y secciones con PERFORM
- Diferenciar PERFORM básico de PERFORM THRU
- Manejar condiciones 88-level para lógica declarativa

## 📚 Contenido

### 1-teoria/

- **verbos-basicos.md**: MOVE (individual y grupal), COMPUTE, DISPLAY, ACCEPT
- **if-evaluate.md**: IF/ELSE/END-IF, EVALUATE/WHEN/WHEN OTHER, condiciones
- **perform-intro.md**: PERFORM párrafo, PERFORM THRU, PERFORM TIMES
- **88-level.md**: Condition names, SET TO TRUE, lógica declarativa
- **operadores.md**: Aritméticos, relacionales, lógicos en COBOL

### 2-practicas/

- **calculadora.cbl**: Operaciones aritméticas con COMPUTE
- **clasificador-edad.cbl**: EVALUATE para clasificar grupos etarios
- **validacion-88.cbl**: Uso de 88-level para estados de cuenta

### 3-proyecto/

- **starter/calculadora-prestamo.cbl**: Calculadora de préstamo bancario con cuota mensual, interés compuesto, validaciones

## ⏱️ Distribución (10h)

- Teoría: 3h | Prácticas: 4h | Proyecto: 3h

## 📌 Entregables

- [ ] Clasificador con EVALUATE para al menos 5 categorías
