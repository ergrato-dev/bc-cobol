# Rúbrica de Evaluación - Semana 04

## Conocimiento 🧠 (30%)

| Criterio | Insuficiente (0-5) | Básico (6-7) | Competente (8-9) | Avanzado (10) |
|----------|-------------------|--------------|------------------|---------------|
| FILE-CONTROL | No configura archivos | Configuración básica | SELECT/ASSIGN correcto | Múltiples archivos |
| OPEN/READ/WRITE | No domina verbos | Verbos básicos sin status | Con FILE STATUS | Manejo completo de errores |
| EOF | No detecta fin de archivo | AT END sin validación | PERFORM UNTIL correcto | Procesamiento robusto |

## Desempeño 💪 (40%)

| Criterio | Insuficiente | Básico | Competente | Avanzado |
|----------|-------------|--------|------------|----------|
| Lectura | No lee archivo | Lee sin FILE STATUS | FILE STATUS correcto | Múltiples archivos |
| Escritura | No escribe | Escribe sin verificar | Verifica escritura | Maneja archivos grandes |
| Flujo | Datos perdidos | Procesa pero con bugs | Funciona correcto | Optimizado |

## Producto 📦 (30%)

**Aprobación**: Mínimo 7/10 en cada tipo de evidencia
