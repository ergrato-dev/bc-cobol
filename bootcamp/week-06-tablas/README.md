# Semana 06: Tablas y Arrays

## 🎯 Objetivos

Al finalizar esta semana, el estudiante será capaz de:

- Declarar tablas con OCCURS clause
- Indexar tablas con INDEXED BY
- Realizar búsqueda secuencial con SEARCH
- Realizar búsqueda binaria con SEARCH ALL (tablas ordenadas)
- Implementar tablas multidimensionales (OCCURS anidados)
- Usar PERFORM VARYING para recorrer tablas

## 📚 Contenido

### 1-teoria/

- **occurs-clause.md**: Declaración, TIMES, DEPENDING ON
- **indexed-by.md**: Índices vs subíndices, USAGE IS INDEX
- **search.md**: SEARCH con AT END/WHEN, condiciones
- **search-all.md**: SEARCH ALL, ASCENDING/DESCENDING KEY, búsqueda binaria
- **multidimensional.md**: OCCURS anidado, tablas 2D y 3D

### 2-practicas/

- **tabla-tasas.cbl**: Tabla de tasas de interés por plazo
- **busqueda-cliente.cbl**: Búsqueda secuencial y binaria en tabla de clientes
- **matriz-tarifas.cbl**: Tabla bidimensional de tarifas por producto y canal

### 3-proyecto/

- **starter/tabla-amortizacion.cbl**: Generar tabla de amortización de préstamo con cálculos por período
