# Semana 07: Manipulación de Strings

## 🎯 Objetivos

Al finalizar esta semana, el estudiante será capaz de:

- Concatenar cadenas con STRING y DELIMITED BY
- Dividir cadenas con UNSTRING y DELIMITED BY
- Inspeccionar y modificar cadenas con INSPECT
- Contar ocurrencias, reemplazar caracteres
- Usar referencia y modificación de datos (reference modification)
- Aplicar funciones intrínsecas de string: LENGTH, TRIM

## 📚 Contenido

### 1-teoria/

- **string.md**: Concatenación, DELIMITED BY SIZE/SPACE/string, POINTER, OVERFLOW
- **unstring.md**: División, DELIMITED BY, TALLYING, POINTER
- **inspect.md**: TALLYING (contar), REPLACING (reemplazar), CONVERTING
- **reference-modification.md**: variable(start:length), acceso parcial
- **funciones-intrinsecas.md**: FUNCTION LENGTH, FUNCTION TRIM, FUNCTION UPPER-CASE

### 2-practicas/

- **formatear-nombre.cbl**: STRING para construir nombre completo
- **parsear-csv.cbl**: UNSTRING para procesar línea CSV
- **limpiar-datos.cbl**: INSPECT para limpiar y normalizar datos

### 3-proyecto/

- **starter/procesador-transacciones.cbl**: Procesar archivo de transacciones bancarias, parsear cada línea con UNSTRING, validar con INSPECT

## ⏱️ Distribución (10h)

- Teoría: 3h | Prácticas: 4h | Proyecto: 3h

## 📌 Entregables

- [ ] Formateo de nombres con STRING
- [ ] Parser CSV con UNSTRING
- [ ] Proyecto de procesador de transacciones completado
