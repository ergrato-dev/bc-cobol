# Semana 09: Reportes Profesionales

## 🎯 Objetivos

Al finalizar esta semana, el estudiante será capaz de:

- Diseñar reportes con WRITE BEFORE/AFTER ADVANCING
- Implementar control de página y salto de línea
- Crear encabezados, detalle y pie de página
- Generar totales y subtotales por grupo
- Usar edición de datos para formato profesional (moneda, fechas)
- Implementar control de ruptura (break control)

## 📚 Contenido

### 1-teoria/

- **write-advancing.md**: BEFORE/AFTER, líneas, PAGE, control de impresora
- **estructura-reporte.md**: Encabezado, detalle, pie, numeración de página
- **totales-subtotales.md**: Acumuladores, ruptura por grupo, RESET
- **edicion-datos.md**: Signo flotante, supresión de ceros, formato moneda
- **control-ruptura.md**: Detección de cambio de grupo, totales por nivel

### 2-practicas/

- **reporte-simple.cbl**: Reporte con encabezado, detalle y pie
- **reporte-agrupado.cbl**: Reporte con subtotales por categoría
- **reporte-formateado.cbl**: Reporte con edición de moneda y fechas

### 3-proyecto/

- **starter/estado-cuenta.cbl**: Generar estado de cuenta bancario mensual con encabezado, movimientos, saldo inicial/final, subtotales por tipo de movimiento

## ⏱️ Distribución (10h)

- Teoría: 3h | Prácticas: 4h | Proyecto: 3h

## 📌 Entregables

- [ ] Reporte simple con control de página
- [ ] Reporte agrupado con subtotales
- [ ] Proyecto de estado de cuenta completado
