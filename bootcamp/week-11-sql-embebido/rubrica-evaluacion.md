# Rúbrica de Evaluación - Semana 11

## Conocimiento 🧠 (30%)

| Criterio | Insuficiente (0-5) | Básico (6-7) | Competente (8-9) | Avanzado (10) |
|----------|-------------------|--------------|------------------|---------------|
| EXEC SQL | No comprende sintaxis | Sintaxis básica | Host variables | SQL dinámico |
| Cursores | No usa cursores | Cursor simple | FETCH en loop | Múltiples cursores |
| Transacciones | Sin control | COMMIT básico | COMMIT/ROLLBACK | Puntos de salvado |

## Desempeño 💪 (40%)

| Criterio | Insuficiente | Básico | Competente | Avanzado |
|----------|-------------|--------|------------|----------|
| Conexión | No conecta | Conexión exitosa | Manejo de errores | Pool de conexiones |
| Consultas | Errores SQL | SELECT correcto | JOINs y filtros | Consultas complejas |
| CRUD | Solo SELECT | INSERT/UPDATE | CRUD completo | Con transacciones |

## Producto 📦 (30%)

**Aprobación**: Mínimo 7/10 en cada tipo de evidencia
